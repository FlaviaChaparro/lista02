Tag's de tabelas em html
Tabelas em HTML
Tag Função
TR Cada linha de uma tabela deve sempre
aparecer entre as marcações. (TR = Table Row)
TD Para a divisão em células das linhas.
TH Usado para definir células que representam cabeçalhos.
Caption Legenda da tabela (permitido somente
uma legenda por tabela)
Thead É utilizado para agrupar o cabeçalho de uma tabela.
Tfoot É utilizado para agrupar o rodapé de uma tabela.
Tbody É utilizado para agrupar o conteúdo
geral (corpo) de uma tabela.
As respectivas colunas apresentam os atributos(tag's) e a funcionalidade do mesmo.
